# 装箱助手-模拟退火-v1

> **技能名称**：packing-optimizer-sa-v1
> **算法**：贪心初始化（SKU集中策略）+ 模拟退火（Simulated Annealing）
> **版本**：v1.0
> **创建日期**：2026-07-23

## 触发条件

当用户提到以下内容时触发本技能：
- "装箱优化"、"装箱方案"、"优化装箱"
- "帮我算装箱"、"跑装箱"
- "模拟退火装箱"
- 上传CSV文件并提到"装箱"

## 使用流程

```
运营操作：
  ① 飞书多维表格填写SKU装箱数据
  ② 飞书导出CSV
  ③ 发给Agent（上传CSV文件）
  ④ 说"帮我跑装箱优化"

Agent执行：
  ① 读取上传的CSV文件
  ② 运行 scripts/optimizer.py 脚本
  ③ 输出装箱方案CSV + 关键指标
  ④ 返回结果给用户
```

## CSV输入格式

运营从飞书导出的CSV，表头格式：

```
SKU编号,产品名称,品类,单件重量(kg),包装长(cm),包装宽(cm),包装高(cm),发货数量,备注
JP-0001,硅胶沥水垫,硅胶,0.15,30.0,20.0,2.0,30,需抽真空
JP-0002,不锈钢量勺5件套,金属,0.2,15.0,10.0,3.0,30,
```

必填字段：SKU编号、单件重量(kg)、包装长(cm)、包装宽(cm)、包装高(cm)、发货数量
选填字段：产品名称、品类、备注

## CSV输出格式

装箱方案CSV，每行是一个SKU在一个箱子里的明细：

```
箱号,箱型,外箱尺寸,空运体积重(kg),SKU编号,产品名称,品类,数量,本箱实重(kg),计费重量(kg),是否抛货,超重,头程运费(¥),单件头程(¥)
BOX-001,4号箱(45×40×40),45×40×40,12.0,JP-0009,硅胶冰格,硅胶,23,14.05,14.05,✅实重,✅,632.25,7.71
```

## 核心脚本

脚本路径：`scripts/optimizer.py`（相对于本技能目录）

运行方式：
```bash
python3 scripts/optimizer.py <input.csv> [参数]
```

可调参数（通过命令行传入）：
- `--max-sku`：每箱最多SKU种类数（默认5）
- `--max-wt`：每箱重量上限kg（默认14.5）
- `--air-price`：空运单价¥/kg（默认45）
- `--iterations`：模拟退火迭代次数（默认100000）
- `--seed`：随机种子（默认42，保证可复现）

## 约束策略

| 约束 | 默认值 | 说明 |
|---|---|---|
| 单箱重量 | ≤15kg（含0.5kg包装）| FBA标准，超过收附加费 |
| 填充率 | ≤90% | 留10%给填充物 |
| 每箱SKU数 | ≤5款 | 仓库操作约束，可调 |
| 箱型 | 1-5号箱 | 排除6-7号（体积重太高）|
| 货物覆盖 | 100% | 所有货物必须装箱 |

## 算法说明

```
第1步：贪心初始化（SKU集中策略）
  - 按SKU分组，同款SKU尽量装到同一个箱子
  - 重轻交替（重货SKU和轻货SKU交替装箱）
  - Best-Fit（放剩余空间最小的箱子）

第2步：模拟退火优化
  - 随机扰动：移动/交换/合并箱子里的货物
  - 接受更好的解，偶尔接受更差的解（跳出局部最优）
  - 逐渐降温收敛
  - 目标函数：总计费重量 + SKU数惩罚

第3步：箱型优化
  - 对每个箱子选计费重量最小的箱型
```
